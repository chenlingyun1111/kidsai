# KidsAI

Children's English learning AI voice chat companion app.

## Tech Stack

- **Mobile**: Flutter (iOS/Android)
- **Backend**: Python FastAPI + WebSocket
- **AI**: Qwen-Max (LLM) + Volcano Engine (STT/TTS)
- **Database**: PostgreSQL + pgvector
- **Animation**: Rive

Project scaffolding in progress.
